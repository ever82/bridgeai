# Story Playbook: VisionShare 内容发现与推荐

**Story**: `AS-VS-002`
**Narrative**: 附近有视觉需求，我判断主人是否适合接单。

## 验收旅程概览

本剧本模拟真实用户 "阿明" 通过 Agent 发现附近视觉任务、从地图浏览、筛选任务、查看详情、接受任务到管理已完成任务的完整闭环旅程。Agent 在旅程中持续辅助主人判断是否适合接单，并在发现异常时给出警告。

## 前置条件

1. 已安装 BridgeAI mobile app (iOS/Android)
2. 已有 BridgeAI 账号并完成基础认证，信用分 >= 600
3. 数据库已迁移并包含测试数据 (附近有若干模拟任务分布在 3km 范围内)
4. 服务端运行: `pnpm dev` (apps/server, apps/mobile)
5. 测试账号: `test+vs002@bridgeai.com` / `password123`
6. 手机定位权限已授权
7. 后台已注入测试数据: 任务 A(0.8km, ¥80)、任务 B(3.2km, ¥150)、任务 C(1.5km, ¥50, 已过期)、任务 D(0.5km, ¥200, 要求信用分 >= 800)
8. 推荐队列中包含 2 条优先推荐任务
