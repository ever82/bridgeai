# Story Playbook: 配置求职者 Agent 画像

**Story**: `JOB-001`
**Narrative**: 求职者配置 Agent，包括技能、经验、期望薪资。

## 验收旅程概览

验证求职者用户能完整配置 Agent 画像，包括技能标签、工作经历、期望薪资等关键信息，使 Agent 能精准匹配职位。

## 前置条件

- 服务已启动
- 数据库已迁移
- 用户已注册并登录
- 用户已创建基础 Agent

## 旅程步骤

### Phase 1: 进入 AgentJob 场景配置

**目的**: 验证求职者能进入场景并开始画像配置

**Step 1.1**: 选择 AgentJob 场景

- 操作: 在 Agent 列表或创建向导中选择"AgentJob"场景
- 预期: 进入求职者画像配置页面

**Step 1.2**: 选择角色身份

- 操作: 选择"我是求职者"
- 预期: 展示求职者专属配置表单

### Phase 2: 填写核心画像信息

**目的**: 验证所有求职者画像字段可正常填写和保存

**Step 2.1**: 技能标签配置

- 操作: 在技能标签区域搜索并添加技能（如"React"、"Python"、"项目管理"）
- 预期: 技能标签正常显示，支持搜索和自定义添加

**Step 2.2**: 工作经历填写

- 操作: 填写最近的工作经历（公司名、职位、时间段、工作内容）
- 预期: 经历条目正常保存并展示

**Step 2.3**: 期望薪资设置

- 操作: 设置期望薪资范围（最低值和最高值）
- 预期: 薪资范围正常保存，Agent 在协商时不会低于底线

**Step 2.4**: 求职偏好设置

- 操作: 设置工作地点偏好、行业偏好、工作类型（全职/兼职/远程）
- 预期: 偏好信息保存成功

### Phase 3: 简历上传与预览

**目的**: 验证简历附件功能和画像预览

**Step 3.1**: 上传简历附件

- 操作: 上传 PDF 格式简历
- 预期: 简历上传成功，Agent 可读取简历内容

**Step 3.2**: 画像预览

- 操作: 点击"预览 Agent 画像"
- 预期: 展示完整的求职者画像卡片，包含所有配置信息

### Phase 4: 异常场景

**目的**: 验证画像配置的边界情况

**Step 4.1**: 必填字段为空提交

- 操作: 不填写关键信息直接保存
- 预期: 表单校验提示必填项

**Step 4.2**: 薪资范围不合理

- 操作: 输入最低薪资高于最高薪资
- 预期: 表单校验提示薪资范围错误
