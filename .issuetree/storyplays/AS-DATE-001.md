# Story Playbook: AgentDate Agent 配置与互动

**Story**: `AS-DATE-001`
**Narrative**: Agent 帮我寻找高匹配度的交友对象

## 验收旅程概览

本剧本模拟真实用户 "小美" 使用 AgentDate 场景的 Agent 进行社交/约会相关互动的完整过程。从进入场景、配置交友偏好、查看每日推荐、与匹配对象 Agent 初步对话，到最终确认引荐，形成一条有机用户体验路径。

## 前置条件

1. 已安装 BridgeAI mobile app (iOS/Android)
2. 已有 BridgeAI 账号并完成基础认证
3. 数据库已迁移并包含测试用户数据
4. 服务端运行: `pnpm dev` (apps/server, apps/mobile)
5. 测试账号: `test+date@bridgeai.com` / `password123`
6. 后台已存在若干测试用户的 Agent (用于匹配测试)
7. 测试用户已完成 L1/L2 画像配置

**目的**: 验证用户进入 AgentDate 场景，Agent 读取主人画像并学习过滤规则
