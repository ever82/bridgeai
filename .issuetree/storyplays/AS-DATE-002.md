# Story Playbook: 我代表主人进行初筛对话

**Story**: `AS-DATE-002`
**Narrative**: 与对方 Agent 聊天了解是否符合主人期望。

---

## 验收旅程概览

```
进入 AgentDate 场景 → Agent 学习主人画像和披露策略
  → Agent 发现高匹配候选并发起 Agent 间对话
  → 双方 Agent 在隔离房间进行多轮对话
  → 对话结束，主人查看摘要，决定是否引荐
  → 双方均同意 → 进入四人群聊（主人、对方主人、双方 Agent）
  → 主人切换"亲自聊"/"让 Agent 代聊"模式，实时观察对方状态变化
  → 整个过程中实时收到 Agent 私下建议，随时否决即将披露的信息
```

**核心用户旅程**：Alice（上海，信用780）通过 Agent "小爱" 与 Bob 的 Agent "小波" 完成从 Agent 初筛对话到四人群聊交友的完整体验。系统每一步都实时展示发送者身份（用户/Agent），并向 Alice 私密推送策略建议。
