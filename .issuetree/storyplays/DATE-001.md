# Story Playbook: 配置交友 Agent 画像

**Story**: `DATE-001` | **Narrative**: 用户配置 AgentDate 场景下的交友偏好，通过向导式配置和 AI 智能提炼完成画像设置

## 验收旅程概览

本剧本模拟真实用户 "小美" 首次使用 AgentDate 配置交友画像的完整流程，涵盖地域过滤、属性过滤、信用过滤三大核心系统的有机验证。

## 前置条件

1. 已安装 BridgeAI mobile app (iOS/Android)
2. 已有 BridgeAI 账号并完成基础认证
3. 数据库已迁移并包含测试用户数据
4. 服务端运行: `pnpm dev` (apps/server, apps/mobile)
5. 测试账号: `test+dating@bridgeai.com` / `password123`
