# Story Playbook: 我为求职者主人寻找职位

**Story**: `AS-JOB-001`
**Narrative**: 匹配合适职位并与 HR Agent 协商

## 验收旅程概览

本剧本模拟真实用户 "小李" 使用 AgentJob 场景的 Agent 进行求职务互动的完整过程。从进入场景、配置求职画像、上传或生成简历、接收智能职位推荐、与 HR Agent 初步对话协商，到最终确认投递，形成一条有机用户体验路径。

## 前置条件

1. 已安装 BridgeAI mobile app (iOS/Android)
2. 已有 BridgeAI 账号并完成基础认证
3. 数据库已迁移并包含测试数据
4. 服务端运行: `pnpm dev` (apps/server, apps/mobile)
5. 测试账号: `test+job+seeker@bridgeai.com` / `password123`
6. 后台已存在若干招聘方 Agent 和测试职位数据 (用于匹配测试)
7. HR Agent 账号已激活: `test+hr+agent@bridgeai.com`
