# Story Playbook: 我为消费者主人比价

**Story**: `AS-AD-001` | **Narrative**: 同时与多个商家 Agent 谈判争取最优价格。

## 验收旅程概览

本剧本模拟真实用户"小王"通过 AgentAd 场景，让自己的消费者 Agent 同时与多个商家 Agent 展开比价谈判的完整旅程。从发起比价请求、商家 Agent 响应报价、消费者 Agent 解析各家供给信息、生成比价报告，到最终锁定最优方案并处理无合适方案的异常情况，形成一条有机的多 Agent 协作体验路径。

## 前置条件

1. 已安装 BridgeAI mobile app (iOS/Android)
2. 已有 BridgeAI 账号并完成基础认证
3. 数据库已迁移 (`pnpm db:migrate`)
4. 服务端运行: `pnpm dev` (apps/server, apps/mobile)
5. 测试账号: `test+consumer@bridgeai.com` (消费者) / `test+merchant1@bridgeai.com` 等 (商家)
6. 后台已存在至少 3 个商家 Agent 发布了有效优惠方案 (用于比价测试)
7. 商家 Agent 已完成 L2 供给信息配置，包含优惠标题、折扣、有效期等
8. 测试用商品: 同一品类商品 (如智能手机、耳机等)，方便横向对比
