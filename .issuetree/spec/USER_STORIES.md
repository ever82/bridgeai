# 用户故事

## US-001: AI 解析需求
**角色**: 普通人（需求方）
**描述**: 用户输入"帮我看看三里屯那边堵车是怎么回事"，AI 自动解析出地点是三里屯、意图是查看堵车情况，然后生成任务单。
**验收标准**:
- [ ] 用户输入自然语言描述
- [ ] 系统返回解析后的地点+意图
- [ ] 系统显示发布的任务

## US-002: 附近接单拍照
**角色**: 外卖骑手（供给方）
**描述**: 骑手打开接单界面，看到附近有"三里屯附近拍照"任务，点击接单，拍照上传，获得10积分。
**验收标准**:
- [ ] 接单界面显示附近可用任务
- [ ] 点击接单后进入拍照界面
- [ ] 拍照并上传后获得积分奖励

## US-003: 查看照片赚积分
**角色**: 普通人（需求方）
**描述**: 用户支付5积分查看一张照片，看到照片后觉得有用，给拍摄者好评。
**验收标准**:
- [ ] 照片列表页显示价格和缩略图
- [ ] 支付积分后显示完整照片
- [ ] 可以对照片评分

## US-004: 历史照片查询
**角色**: 普通人（需求方）
**描述**: 用户想知道"去年五一这家医院人多不多"，输入后看到当时的历史照片。
**验收标准**:
- [ ] 支持按地点和时间搜索
- [ ] 返回匹配的历史照片
- [ ] 显示拍摄时间
