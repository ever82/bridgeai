# 设计系统

## 1. 颜色规范

### 1.1 主色调

```
┌─────────────────────────────────────────────┐
│  Primary    #6366F1  (靛蓝)  主要操作按钮   │
│  Secondary  #8B5CF6  (紫)   次要操作       │
│  Success    #10B981  (绿)   成功状态       │
│  Warning    #F59E0B  (橙)   警告提示       │
│  Danger     #EF4444  (红)   错误/拒绝      │
│  Info       #3B82F6  (蓝)   信息提示       │
└─────────────────────────────────────────────┘
```

**使用场景**:
- Primary: 主要按钮、选中状态、强调文字

- Warning: 警告消息、需要关注的内容
- Danger: 错误消息、删除操作、危险操作
- Info: 提示信息、系统通知

## 2. 字体规范

### 2.1 字体栈

```
中文: -apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif
英文: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
等宽: "SF Mono", "Monaco", "Inconsolata", monospace
```

### 2.2 字号层级

| 层级 | 字号 | 行高 | 字重 | 用途 |
|------|------|------|------|------|
| H1 | 28px | 1.3 | 700 | 页面大标题 |
| H2 | 20px | 1.4 | 600 | 区块标题 |
| H3 | 16px | 1.4 | 600 | 卡片标题 |
| Body | 14px | 1.6 | 400 | 正文 |
| Caption | 12px | 1.5 | 400 | 辅助文字 |
| Small | 12px | 1.5 | 400 | 标签、说明 |

### 2.3 字体缩放支持

- 支持系统字体大小设置
- 最小保证可读性：Body ≥ 12px
- 最大支持 200% 缩放

## 4. 间距规范

### 4.1 基础单位

基础间距单位: 4px

### 4.2 间距层级

| 名称 | 数值 | 用途 |
|------|------|------|
| xs | 4px | 紧凑间距、图标与文字 |
| sm | 8px | 默认间距、元素之间 |
| md | 16px | 区块间距、卡片内边距 |
| lg | 24px | 大区块间距 |
| xl | 32px | 页面间距 |
| 2xl | 48px | 大模块间距 |

### 4.3 组件间距

- 卡片内边距: 16px
- 列表项间距: 12px
- 按钮内边距: 12px 24px
- 输入框高度: 44px (触摸友好)
- 底部导航高度: 56px
- 顶部状态栏高度: 56px

### 4.4 安全区

- iOS底部安全区: 34px (iPhone X+)
- Android导航栏: 48px
- 顶部状态栏高度: 44px (iOS) / 24px (Android)

## 7. 按钮规范

### 7.1 按钮类型

**主要按钮 (Primary)**:
- 背景: Primary (#6366F1)
- 文字: 白色
- 圆角: 8px
- 高度: 44px
- 内边距: 12px 24px

**次要按钮 (Secondary)**:
- 背景: Surface
- 边框: 1px solid Border
- 文字: TextPrimary
- 圆角: 8px

**幽灵按钮 (Ghost)**:
- 背景: 透明
- 文字: Primary
- 无边框

**危险按钮 (Danger)**:
- 背景: Danger (#EF4444)
- 文字: 白色

### 7.2 按钮状态

- 默认: 正常状态
- Hover: 亮度降低 10%
- Active: 亮度降低 20%
- Disabled: 透明度 50%
- Loading: 显示加载动画

## 10. 响应式断点

```
Mobile:     < 768px   单列布局

Desktop:    > 1024px   三列布局
```

### 10.1 移动端

- 单列布局
- 底部导航
- 全屏页面
- 抽屉式侧边栏

### 10.2 平板端

- 双列布局
- 左侧固定导航
- 卡片网格

### 10.3 桌面端

- 三列布局
- 左侧导航 + 中间内容 + 右侧辅助
- 快捷键支持
- 更大的点击区域

## 11. 动效规范

### 11.1 时长

- 快速: 100ms (按钮点击、小状态变化)
- 正常: 200ms (切换、展开)
- 慢速: 300ms (页面切换、大动画)

### 11.2 缓动函数

```
Ease:       cubic-bezier(0.4, 0, 0.2, 1)
EaseIn:     cubic-bezier(0.4, 0, 1, 1)
EaseOut:    cubic-bezier(0, 0, 0.2, 1)
EaseInOut:  cubic-bezier(0.4, 0, 0.2, 1)
Spring:     cubic-bezier(0.34, 1.56, 0.64, 1)
```

### 11.3 常用动效

**页面切换**:
- 时长: 300ms
- 缓动: EaseInOut
- 效果: 新页面从右侧滑入

**消息出现**:
- 时长: 200ms
- 缓动: EaseOut
- 效果: 从底部滑入 + 渐显

**人机切换**:
- 时长: 300ms
- 缓动: Spring
- 效果: 缩放 + 状态变化

**Agent建议气泡**:
- 时长: 200ms
- 缓动: EaseOut
- 效果: 从右侧滑入

## 12. 无障碍设计

### 12.1 颜色对比度

- 正文对比度: ≥ 4.5:1 (WCAG AA)
- 大文字对比度: ≥ 3:1
- 界面元素对比度: ≥ 3:1

### 12.2 触摸目标

- 最小尺寸: 44px × 44px
- 推荐尺寸: 48px × 48px
- 间距: ≥ 8px

### 12.3 屏幕阅读器

- 所有图标提供 aria-label
- 重要状态变化使用 aria-live
- 复杂组件提供 role 属性

### 12.4 键盘导航

- Tab键顺序符合视觉顺序
- 支持 Enter/Space 激活
- Esc 关闭弹窗/菜单
- 焦点状态清晰可见
