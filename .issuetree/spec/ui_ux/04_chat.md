## 2. 群聊成员区域

### 2.1 展开状态（默认）
- 显示4个成员头像网格
- 每个成员显示：头像、名称、角色标签、在线状态
- 角色标签：Agent/真人/亲自聊/代聊
- 在线状态：🟢在线 🟡Away ⚪离线

### 2.2 折叠状态
- 点击标题栏折叠
- 仅显示一行：当前谁在聊天
- 释放更多空间给消息区域

## 4. Agent私聊建议（方案1：悬浮气泡）

```
┌─────────────────────────────────────────────┐
│                                             │
│  🤖 小A (Agent)              14:32          │
│  ┌─────────────────────────────────┐       │
│  │ 你好，我是小A...                │       │
│  └─────────────────────────────────┘       │
│                                             │
│                  ┌───────┐                  │
│                  │ 💡    │                  │
│                  │ 小A   │                  │
│                  │ 建议  │                  │
│                  └───────┘                  │
│                                             │
│  👤 小红 (亲自)                            │
│  ┌─────────────────────────────────┐       │
│  │ 你好，我是小红...               │       │
│  └─────────────────────────────────┘       │
│                                             │
│          ┌──────────────────┐              │
│          │  🤖 小A对你说:   │              │
│          │                  │              │
│          │ "对方态度友好，  │              │
│          │  可以深入交流"   │              │
│          │                  │              │
│          │ [采用]  [忽略]   │              │
│          └──────────────────┘              │
│                                             │
└─────────────────────────────────────────────┘
```

**气泡交互说明**:
1. **触发时机**: Agent有新建议时，右侧出现 💡 图标
2. **展开方式**: 点击 💡 图标，气泡从右侧滑入显示建议内容
3. **自动消失**: 5秒无操作自动收起，图标保持显示
4. **操作选项**: 
   - [采用]: 将建议填入输入框
   - [忽略]: 收起气泡，不再提示此建议
   - [设置]: 进入"建议偏好"设置

**建议频率控制**:
- 默认: 仅在关键节点提示（对方切换状态、Agent认为需要人工介入）
- 可设置: 关闭建议 / 精简模式 / 详细模式

## 6. 消息操作

### 6.1 长按消息菜单

```
┌─────────────────────────────────────────────┐
│                                             │
│  ┌─────────────────────┐                   │
│  │  复制               │                   │
│  │  回复               │                   │
│  │  转发               │                   │
│  │  撤回（2分钟内）    │                   │
│  │  ────────────────   │                   │
│  │  举报               │                   │
│  └─────────────────────┘                   │
│                                             │
└─────────────────────────────────────────────┘
```

### 6.2 撤回消息
- 发送后2分钟内可撤回
- 撤回后显示: "你撤回了一条消息"
- 对方看到的是: "对方撤回了一条消息"

### 6.3 消息状态
- 单勾: 已发送
- 双勾: 已送达
- 双勾蓝: 已读
