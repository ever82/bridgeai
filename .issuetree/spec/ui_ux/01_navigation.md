# 导航设计

## 1. 底部导航栏（移动端）

```
┌─────────────────────────────────────────────┐
│                                             │
│           [当前页面内容区域]                   │
│                                             │
│                                             │
├─────────────────────────────────────────────┤
│  🏠      🎯      📝      💬      👤        │
│ 首页    场景    任务    消息    我的         │
│            [未读角标]  [红点]                │
└─────────────────────────────────────────────┘
```

**导航项说明**:
- **首页**: 个人 Agent 状态、信用分、积分、推荐匹配
- **场景**: 四大场景入口（VisionShare / AgentDate / AgentJob / AgentAd）
- **任务**: 跨场景的任务聚合中心（我发布的、我接单的、进行中的）
- **消息**: 所有群聊会话列表、系统通知
- **我的**: 个人资料、Agent 配置、设置、隐私

**设计要点**:
- 底部导航固定 5 项，点击区域 ≥ 56px 高度
- 当前选中项使用主色填充，未选中使用灰色线条图标
- 消息、任务页有未读时显示红点/数字角标

---

## 2. 顶部状态栏

```
┌─────────────────────────────────────────────┐
│ BridgeAI  ≡                    🔍    🔔  💎 │
│                                             │
│  [页面标题]                          [操作]  │
└─────────────────────────────────────────────┘
```

**状态栏元素**:
- 左侧: Logo + 汉堡菜单（展开侧边导航）
- 右侧: 搜索、通知（红点提示）、积分余额

**搜索入口**:
- 点击搜索图标进入搜索页
- 支持搜索：用户、需求、任务、历史会话
- 提供搜索历史、热门推荐

## 4. 响应式导航

### 平板端 (768px - 1024px)
- 侧边固定导航栏替代底部导航
- 左侧显示导航，右侧显示内容

### 桌面端 (> 1024px)
- 左侧固定导航栏
- 顶部简化状态栏（仅显示通知、积分）
- 支持快捷键导航
