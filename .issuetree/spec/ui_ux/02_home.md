## 2. 功能模块说明

### 2.1 个人信息卡片
- 显示头像、昵称、信用分、积分
- 点击可进入个人资料编辑
- "配置我的 Agent"快捷入口

### 2.2 Agent 状态卡片
- 显示当前 Agent 工作状态
- 列出各场景的进行中任务数量
- 点击跳转任务详情

### 2.3 今日推荐
- 基于位置和偏好推荐需求/匹配
- 显示关键信息（距离、奖励、匹配度）
- 一键响应操作

### 2.4 快速入口
- 四大场景的快捷入口
- 点击直达对应场景首页
