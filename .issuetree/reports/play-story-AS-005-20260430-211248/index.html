6. Critic 疑点 — 严重 (Critical) — 7 项

⚠️ 以下问题影响 Story 核心功能的真实性，必须修复：

CRITICAL AS-005-AC-2
Agent 自主发起对话服务从未被任何外部代码调用 — 是死代码
位置: apps/server/src/services/dating/agentInitiatedChat.ts:90
initiateChat() 是 Story 的核心 API，但全代码库（除自身模块、index.ts re-export、单元测试外）没有 caller。Socket handler、REST 路由、job queue 均无入口触发，意味着 Agent 自主决定与候选 Agent 发起对话在生产路径上根本不会执行，仅完成签名层验证。
建议: 增加调度层（cron / 候选生成回调 / Socket 事件 / worker），在候选列表生成后自动调用 initiateChat。
CRITICAL ISSUE-AI004~c1
Agent 自动响应是硬编码模板字符串，没有调用 LLM
位置: apps/server/src/services/dating/agentInitiatedChat.ts:335-363
generateResponseContent 完全使用 4-5 条候选模板字符串数组按 turnIndex % responses.length 轮播。没有 LLM 调用、没有上下文管理、没有人格化、没有意图识别。剧本要求的 5-10 轮自然对话评估性格契合度实际只是机械复读。
建议: 替换为 agentDialogService.generateMessage() 真实 LLM 调用，并接入人格 prompt 与上下文管理。
CRITICAL US-AGENT-009~func
agent:private_advice 私聊建议通道在代码中完全不存在
位置: apps/server (全代码库 grep)
剧本明确声明 agent:private_advice 通道仅推送给主人 socket，但全代码库 grep private_advice|privateAdvice|私聊建议 在 server 与 mobile 均无任何匹配。AS-008 整套实时分析意图、发送私下建议、风险提示、话题推荐、一键采纳在 server 端无 emit 事件，mobile 无订阅与 UI。
