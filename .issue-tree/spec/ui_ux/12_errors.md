## 8. 错误码规范

| 错误码 | 说明 | 用户提示 |
|--------|------|----------|
| ERR_NETWORK | 网络连接失败 | 网络连接失败，请检查网络设置 |
| ERR_TIMEOUT | 请求超时 | 请求超时，请稍后重试 |
| ERR_SERVER_500 | 服务器错误 | 系统繁忙，请稍后重试 |
| ERR_SERVER_503 | 服务不可用 | 系统维护中，请稍后再试 |
| ERR_AUTH_401 | 未授权 | 登录已过期，请重新登录 |
| ERR_AUTH_403 | 禁止访问 | 无权限执行此操作 |
| ERR_NOT_FOUND | 资源不存在 | 内容不存在或已被删除 |
| ERR_VALIDATION | 参数错误 | 请检查输入内容 |
| ERR_CONFLICT | 操作冲突 | 该内容已被修改，请刷新后重试 |
| ERR_RATE_LIMIT | 请求过于频繁 | 操作过于频繁，请稍后再试 |
