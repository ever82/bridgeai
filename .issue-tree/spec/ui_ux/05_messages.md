## 2. 会话列表项规范

### 2.1 布局结构
```
┌─────────────────────────────────────┐
│  [头像]  [场景标签]  [时间]          │
│          [状态标签]                   │
│          [最后消息预览]               │
│                                     │
└─────────────────────────────────────┘
```

### 2.2 状态标签
- **对方亲自聊中**: 👤 + 文字，绿色
- **Agent协商中**: 🤖 + 文字，蓝色
- **等待回复中**: ⏳ + 文字，灰色
- **已结束**: ✓ + 文字，灰色

### 2.3 未读提示
- 头像右下角显示红色圆点
- 消息预览文字加粗
- 时间显示为红色

## 4. 消息操作

### 4.1 左滑操作
- 左滑会话项显示快捷操作
- [标为已读] [置顶] [删除]
