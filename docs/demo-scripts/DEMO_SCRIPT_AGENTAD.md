# AgentAd Demo Script

**场景**: AgentAd - AI 广告匹配，让用户和商家通过 AI Agent 实现精准营销和优惠协商。

**演示时长**: 3-5 分钟
**演示账号**: `demo-ad@bridgeai.com` / `Demo1234!` (David，商家视角)
**演示环境**: 需预先 Seed 数据，见 `docs/DEMO_DATA.md`

---

## 场景概览

- 商家 David 配置商家 Agent（David's Coffee，新用户首杯半价活动）
- 系统根据用户偏好自动匹配目标客户
- 用户和商家 Agent 进行优惠协商
- 用户获得更优惠的价格，商家获得精准客户

---

## 操作步骤与讲解词

### Step 1: 配置商家 Agent 画像 (0:00 - 1:30)

**操作**:

1. 登录 `demo-ad@bridgeai.com`（David）
2. 进入 AgentAd 场景
3. 点击"配置商家 Agent"
4. 设置商家信息：David's Coffee，连锁精品咖啡品牌
5. 设置优惠活动：新用户首杯半价（原价 30 元，活动价 15 元）
6. 设置目标客户画像：咖啡爱好者、25-40岁、办公区域用户
7. 设置推广预算：每周 1000 积分用于广告投放

**讲解词**:

> AgentAd 让商家的营销更精准、更高效。
> David 只需要设置他的咖啡店信息和优惠活动。
> 他的目标客户是：咖啡爱好者、办公区域、25-40岁人群。
> AI Agent 会自动帮他找到最合适的潜在客户，不再是广撒网式的广告。

---

### Step 2: 查看用户匹配情况 (1:30 - 2:30)

**操作**:

1. 进入"投放效果"页面
2. 看到匹配统计：已匹配 128 位潜在客户
3. 显示客户画像分布（年龄、兴趣、位置）
4. 查看单个用户的匹配理由

**讲解词**:

> 商家可以实时查看投放效果。
> 系统已自动匹配了 128 位潜在客户，并展示画像分布。
> 每个用户的匹配理由清晰展示：为什么这个用户会感兴趣。
> 商家可以据此调整目标客户画像，优化投放策略。

---

### Step 3: Agent 精准推送与协商 (2:30 - 3:30)

**操作**（切换到用户视角）:

1. 用户 Alice 收到推送通知："附近有咖啡店新用户优惠"
2. Alice 点击查看，AI 推荐理由：咖啡爱好者、附近工作
3. Alice 表达兴趣："我想要更低的折扣"
4. Alice 的 Agent 与商家 Agent 开始协商

**讲解词**:

> 对于用户，AgentAd 是被动接收转为主动服务。
> Alice 收到了她真正感兴趣的广告，而不是垃圾信息。
> 当 Alice 表示想要更优惠的价格，她的 Agent 会代表她与商家 Agent 谈判。
> 这是一个双赢的过程：用户获得更好的 deal，商家获得一个明确有购买意向的客户。

---

### Step 4: 四人群聊优惠成交 (3:30 - 4:30)

**操作**:

1. 进入四人群聊：Alice、David、以及双方的 Agent
2. 观察 Agent 间协商：
   - Alice 的 Agent: "Alice 是重度咖啡用户，每周消费 5 杯以上，希望能拿到 7 折。"
   - David's Agent: "7 折接近我们的成本线，但如果是会员可以给 8 折 plus 免费加 shot。"
3. Alice 切换到"亲自聊"，确认下单
4. 点击"一键购买"，跳转商家页面

**讲解词**:

> 优惠协商进入四人群聊，双方 Agent 代表各自立场谈判。
> Alice 的 Agent 说明了她的消费能力，David 的 Agent 给出了合理方案。
> Alice 确认后可以一键购买，跳转到商家页面，自动应用优惠码。
> 整个过程：精准推送 -> 智能协商 -> 无缝购买，用户体验流畅。

---

### Step 5: 购买后评价 (4:30 - 5:00)

**操作**:

1. Alice 完成购买后，弹出评价界面
2. 对商家评分（产品质量、服务态度、优惠真实性）
3. 提交评价

**讲解词**:

> 购买完成后，用户可以对商家评分。
> 评价包括：产品质量、服务态度、优惠是否真实。
> 高评分商家会获得更多推荐机会，低评分商家曝光降低。
> 这形成了一个正向循环，激励商家提供真实优惠。

---

## 转场设计

| 转场点   | 描述                                   |
| -------- | -------------------------------------- |
| 开场     | 展示 BridgeAI 首页，聚焦 AgentAd 卡片  |
| Step 1-2 | 配置完成后，自动刷新投放效果面板       |
| Step 2-3 | 点击"查看详情"，切换到用户视角推送通知 |
| Step 3-4 | 用户表示感兴趣后，触发群聊协商转场     |
| Step 4-5 | 一键购买后，淡入评价界面               |
| 结尾     | 展示 ROI 统计，强调 AI 精准营销的价值  |

---

## 注意事项

- 演示前运行 `npm run db:seed:demo` 初始化数据
- AgentAd 需要展示双向视角：商家侧和用户侧
- 优惠协商是 AgentAd 的亮点，建议在演示中充分展示 Agent 谈判过程
- 一键购买流程需要提前配置好测试商家的页面跳转链接
- 如时间紧张，可跳过评价环节，直接进入总结
