# AgentJob Demo Script

**场景**: AgentJob - AI 求职匹配，让求职者和招聘方通过 AI Agent 自动匹配并完成薪资协商。

**演示时长**: 4-6 分钟
**演示账号**: `demo-job@bridgeai.com` / `Demo1234!` (Carol，求职者视角)
**演示环境**: 需预先 Seed 数据，见 `docs/DEMO_DATA.md`

---

## 场景概览

- 求职者 Carol 配置 Agent 画像（6年React全栈经验）
- 招聘方发布职位需求（招聘高级前端工程师）
- 双方 Agent 自动匹配简历和职位
- 匹配成功后进入四人群聊进行薪资协商
- 达成一致后安排面试

---

## 操作步骤与讲解词

### Step 1: 配置求职者 Agent 画像 (0:00 - 1:30)

**操作**:

1. 登录 `demo-job@bridgeai.com`（Carol）
2. 进入 AgentJob 场景
3. 点击"配置求职 Agent"
4. 设置技能标签：React, TypeScript, Node.js
5. 设置工作经历：6年全栈开发，曾在 XX 公司担任技术负责人
6. 设置期望薪资：30K-45K，偏好远程工作
7. 设置偏好：希望团队技术氛围好，使用 TypeScript

**讲解词**:

> 在 AgentJob 场景，求职者只需要告诉 AI Agent 你的技能和期望。
> Carol 是 6 年经验的 React 全栈工程师，期望 30K 到 45K 的月薪。
> Agent 会根据这些信息自动匹配合适的职位，不再需要海投简历。

---

### Step 2: 查看匹配推荐 (1:30 - 2:30)

**操作**:

1. 进入"职位推荐"页面
2. 看到推荐列表，第一个推荐：某公司招聘高级前端工程师
3. 显示匹配度：87%
4. 显示关键匹配点：React + TypeScript 高度匹配，薪资范围重叠

**讲解词**:

> Carol 的 Agent 自动扫描所有发布的职位。
> 这里推荐了一个高级前端工程师岗位，匹配度 87%。
> 关键匹配点清晰展示：React 技能匹配、TypeScript 要求符合、薪资范围有重叠。
> 用户不需要自己判断，AI 已经做好了筛选。

---

### Step 3: Agent 间简历筛选 (2:30 - 3:30)

**操作**:

1. 点击"申请"按钮
2. 进入 Agent 间沟通界面
3. 观察双方 Agent 自动交流：
   - 招聘方 Agent: "我们正在招聘高级前端工程师，请问您的 TypeScript 经验有多少年？"
   - Carol 的 Agent: "Carol 有 3 年 TypeScript 经验，主要用于 React 全栈开发。"
   - 招聘方 Agent: "很好，我们团队也在用 React + TypeScript，请问您的期望薪资范围？"
   - Carol 的 Agent: "30K 到 45K，偏好有一定远程工作灵活性。"
4. 双方 Agent 互相确认关键信息

**讲解词**:

> 这是 AgentJob 的核心创新：Agent 间自动完成简历筛选和初步沟通。
> 招聘方 Agent 会询问关键问题，求职者 Agent 代表用户作答。
> 双方 Agent 互相验证关键信息（技能、薪资期望等）是否匹配。
> 整个过程保护用户隐私，详细简历仅在双方同意后共享。

---

### Step 4: 四人群聊薪资协商 (3:30 - 5:00)

**操作**:

1. 双方 Agent 确认匹配后，系统提示"进入薪资协商"
2. 进入四人群聊：Carol、招聘方 HR、以及双方的 Agent
3. 观察 Agent 间协商：
   - 招聘方 Agent: "我们预算范围内可以给到 35K，但 Carol 的 Agent 提出了 40K 的期望。"
   - Carol 的 Agent: "Carol 的市场价值是 42K，但如果团队技术挑战足够，可以考虑 38K。"
4. Carol 切换到"亲自聊"，直接与 HR 沟通
5. 双方达成一致：38K + 远程工作灵活性

**讲解词**:

> Agent 间简历筛选通过后，进入薪资协商阶段。
> 四人群聊中，两个 Agent 会代表各自立场进行专业谈判。
> Carol 可以随时接管对话，亲自与 HR 沟通。
> 我们的 AI 还会提供私聊建议，帮助 Carol 评估 offer 是否合理。
> 最终双方达成一致，安排后续面试。

---

### Step 5: 面试后评分 (5:00 - 6:00)

**操作**（模拟面试结束后的评分流程）:

1. 点击"面试完成"
2. 弹出双向评分界面
3. 对招聘方评分（专业度、沟通效率、守信度）
4. 提交评分

**讲解词**:

> 面试结束后，双方可以互相评分。
> 对于求职者，评估公司是否专业、沟通是否高效、是否守信。
> 评分影响信用分，高信用用户在后续求职中获得更好推荐。

## 转场设计

| 转场点   | 描述                                         |
| -------- | -------------------------------------------- |
| 开场     | 展示 BridgeAI 首页，聚焦 AgentJob 卡片       |
| Step 1-2 | 画像配置完成后，自动刷新推荐列表             |
| Step 2-3 | 点击申请后，切换到 Agent 对话界面            |
| Step 3-4 | Agent 确认匹配后，显示"进入薪资协商"转场动画 |
| Step 4-5 | 用户点击"达成一致"，淡入面试安排界面         |
| 结尾     | 展示简历曝光统计，强调 AI 匹配效率           |

---

## 注意事项

- 演示前运行 `npm run db:seed:demo` 初始化数据
- 招聘方 Agent 需要另一个账号，可使用 `demo-admin@bridgeai.com` 模拟
- Agent 间对话可以加速播放，重点展示关键问答环节
- 薪资协商是 AgentJob 的亮点，建议在演示中充分展示
- 如时间紧张，可跳过评分环节，直接进入总结
